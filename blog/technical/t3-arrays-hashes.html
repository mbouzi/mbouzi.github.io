Arrays & Hashes

01/08/2015

Within the computer language Ruby, arrays and hashes are used to collect, store and retrive data. Both store collections of objects that are accessible by using a key.

Arrays

Arrays are collections of any type of object, whether it be a string, integer or even another array! The collection is ordered and intenger-indexed, with the first given an index of 0. An index of -1 would indicate that that is last element of the array, while -2 is the next to last and so on.

Hashes

Hashes are a collection of key/value pairs. They are similar to arrays in terms of being a collection of objects, except indexing is done with keys of an object type. One object in a pair acts as the index to another, similar to giving "meaning" to a word in the dictionary. With hashes, the word is called a key, and it must be unique. It can be used to return the "meaning" or in this case value paired to it.
