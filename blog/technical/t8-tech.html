What is Ruby on Rails?

02/15/2015

Ruby on Rails is a software library that extends the Ruby programming language. Ruby is known among programmers for its ease of use and uncluttered syntax. It is a framework used to build websites. Often called simply Rails, Ruby on Rails is software code that is added to Ruby.

Popular among web startups, Rails combines Ruby with HTML, CSS, and JavaScript to create a web application that runs on a web server. Since it runs on a web server, Rails is considered to be a back-end web application development platform.
