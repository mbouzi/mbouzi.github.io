Classes vs IDs

12/31/2014

CSS selectors allow one to select and manipulate HTML elements based on things attributes classes and IDs. Class and ID selectors are bascially ways of describing content in an HTML document. They can be used to categorize elements that are similar stylistically or set an element apart from others of the same type. These tools are especially useful when styling and manipulating in CSS.

Classes

Classes are essentially used to group similar elements together. The same class can be used on multiple elements and multiple classes can be used on the same element. For example, styling information than needs to be applied to multiple objects on a page should be done by appling a class.

IDs

IDs are given to an element to make it unique. A particular ID should only be given to one element, and an element can only have one ID. Your code will not pass validation if the same ID is used on more than one element.

--Both an ID and multiple classes can be used on a single element. Regarding CSS, there is nothing than can be done with and ID that can't be done with a class and vice versa.--
